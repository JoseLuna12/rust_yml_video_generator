#### Rust Video Generator

## yaml file

```yaml
title: Chill Music
description: description of the video
lang: en
bg:
  title: "Ocean"
  author: "author"
  path: "assets/ocean.mp4"
music:
  - title: "video title"
    author: "music author"
    path: "assets/bounce.mp3"
```
## Proyect
- install ffmpeg
- change path to yaml file in `main.rs`
- cargo run
